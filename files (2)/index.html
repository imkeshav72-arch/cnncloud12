⚙ Backend Setup Guide

Run the Flask CNN API on your local machine

📦 Installation Steps

STEP 1 — Install Python Requirements

# Make sure Python 3.9+ is installed
pip install -r requirements.txt

# Or install manually:
pip install flask flask-cors tensorflow pillow

STEP 2 — Start the Flask Server

# Navigate to the backend folder
cd backend

# Run the server (auto-downloads model ~14MB on first run)
python app.py

# You should see:
# ✅ Model loaded successfully!
# 🚀 Running on http://localhost:5000

STEP 3 — Open This Frontend

# Open index.html in your browser
# Make sure API URL is set to:
http://localhost:5000

# Click "Test Connection" to verify
# Then upload any image and classify!

ℹ About CloudVision CNN

Cloud-Based Image Classification using Convolutional Neural Network

🎯 Project Overview

This system uses MobileNetV2 — a lightweight CNN architecture pretrained on ImageNet — to classify any image into one of 1000 categories with high accuracy in real-time.

Top-1 Accuracy 71.3%
Top-5 Accuracy 90.1%
Model Size 14 MB
Parameters 3.4 Million
